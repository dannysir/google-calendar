### 체크리스트

---

- redux를 용한 전역 상태 관리.
- date picker 추가.
  - 선택 날짜 상태 관리.
  - 선택 주(week) 상태 관리.
- 주별 보기 달력.
  - date picker에 고른 날짜에 따라 주별 달력 변경.
  - 헤더 버튼 클릭시 주별 달력 변경.
  - 이벤트 표시.
  - 월별 보기 (optional)
  - 달력 클릭 & 드래그로 이벤트 추가. (optional)
- 모달.
  - 좌측 버튼 클릭시 이벤트 추가(제목, 시간 입력).
  - 주별 달력에 이벤트 클릭시 이벤트 수정(수정, 삭제).
  - 반복 일정 (optional)
  - 사용자 인터렉션에 따라 modal 생성 위치.
- 구글 캘린더와 최대한 유사하게 디자인 (optional)
  - y축 라벨 수정.
  - 격자 수정.
  - 헤더 디자인 수정.
